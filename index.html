PVPScores

A Minecraft plugin for managing PVP Kill Scores. Written in JavaScript using ScriptCraft.

A plugin that give you an easy way to manage PVP matches and scoreboards.

Dependencies

ScriptCraft
This plugin (like ScriptCraft) works with any Bukkit/Spigot server.

Installation

Install ScriptCraft, download the latest PVPScoresPlugin release from here, and extract the zip folder to your ScriptCraft plugin directory.

Usage

All of the commands are ops-only (for good reason!) Here is a (sort of) up-to-date list of commands:
startMatch()
endMatch()
resetScores()
